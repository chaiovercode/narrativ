# Gallery

View, manage, and share your generated stories.

## Accessing the Gallery

Click **Gallery** in the navigation bar to view all your generated content.

## Gallery Features

### Story Organization

Stories are organized by:
- **Date**: Most recent first
- **Topic**: The original topic/title
- **Number of slides**: Visual indicator

### Viewing Stories

Click any story to:
- View all slides in full resolution
- Navigate between slides
- See generation details

### Lightbox Mode

Click an individual image to open the lightbox:
- Full-screen preview
- Zoom capability
- Navigate between images
- Direct download option

## Managing Content

### Downloading

#### Single Image
1. Click the image in gallery
2. Click **Download** button
3. Image saves to your Downloads folder

#### Entire Story
1. Open the story
2. Click **Download All**
3. ZIP file downloads with all images

### Deleting Stories

To remove a story:
1. Hover over the story card
2. Click the delete icon
3. Confirm deletion

:::warning
Deletion is permanent. Downloaded copies are not affected.
:::

## Storage Location

Generated stories are stored in:

```
~/Library/Application Support/com.narrativ.app/generated/
```

Each story has its own folder named:
```
{Topic}_{YYYYMMDD_HHMMSS}/
├── 01_slide.png
├── 02_slide.png
├── 03_slide.png
└── ...
```

### Managing Storage

Over time, generated images can accumulate. To free space:

1. Delete old stories from the Gallery
2. Or manually remove folders from the storage location

## Sharing Stories

### For Instagram Stories

1. Download individual images
2. AirDrop to your iPhone, or
3. Upload to iCloud Photos
4. Post from Instagram app

### For Other Platforms

Stories work great on:
- WhatsApp Status
- Facebook Stories
- LinkedIn
- Twitter/X
- TikTok

### Export Tips

- **Story format (9:16)**: Instagram, WhatsApp, TikTok, Facebook Stories
- **Square (1:1)**: Instagram feed, Twitter, LinkedIn
- Use the ZIP download for batch uploads
- Images are already optimized for social media

## Tips

### Organizing Your Work

- Use descriptive topics - they become folder names
- Regular cleanup keeps gallery manageable
- Favorite stories by keeping them, delete experiments

### Quick Access

- Recent stories appear first
- Use browser forward/back for navigation
- Gallery persists between sessions

### Batch Operations

Currently, deletion is per-story. For bulk operations:
- Use Finder to access the storage folder
- Delete multiple story folders at once
- Refresh Gallery to see changes
